A402 Embed Test Page

This simulates a third-party website embedding A402 paywalled content via embed.js

1. Standard Embed (dark theme, auto)

The simplest integration — just a div with vault + resource attributes.

<div class="a402-content"
     data-vault="0x99848affd96be31241c14700e30342152afd31f1"
     data-resource="video-qmeiotst"></div>
<script src="embed.js"></script>

2. Light Theme

Force light theme with data-theme="light".

3. Multiple Widgets

Multiple widgets on the same page — each is independent.

4. Error State (non-existent resource)

Graceful error when the resource doesn't exist on-chain.
